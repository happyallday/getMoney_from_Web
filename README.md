# Welfare Helper - 合规福利获取助手

一个基于浏览器扩展的合规福利获取助手，帮助用户更高效地获取各平台福利，支持表单自动填充和智能提醒功能。

## 🌟 项目简介

**项目名称**: WelfareHelper（福利助手）
**技术架构**: 浏览器扩展 + Python系统服务 + SQLite
**合规策略**: 表单自动填充 + 智能提醒，用户手动确认操作

### 支持平台
- ✅ 三大运营商小程序（移动、联通、电信）
- ✅ 京东（京豆、签到等）
- ✅ 淘宝/天猫（淘金币、积分等）
- ✅ 拼多多（签到、优惠券等）

### 核心功能
- 🤖 **表单自动填充**: 智能检测并自动填充福利页面表单
- 🔔 **智能提醒**: 及时通知福利活动和最佳操作时机
- 📱 **平台适配**: 针对各平台的专门优化适配器
- 💾 **数据统计**: 记录每日福利获取情况和积分统计
- 🛡️ **安全合规**: 所有数据本地存储，完全符合平台规则

### 技术栈
- **浏览器扩展**:原生JavaScript + Chrome Extensions API (Manifest V3)
- **后端服务**: Flask + APScheduler + SQLAlchemy + SQLite
- **数据加密**: Cryptography 42.0
- **系统服务**: pywin32 (Windows), python-daemon (Linux)

## 🚦 快速开始

### 安装依赖
```bash
# 安装Python依赖
pip install -r requirements.txt

# 初始化数据库
python python-backend/models/__init__.py
```

### 开发模式启动
```bash
# 启动Python后端服务
python python-backend/main.py --dev

# 在浏览器中加载扩展
1. 打开Chrome/Edge
2. 访问 chrome://extensions/
3. 启用开发者模式
4. 加载已解压的扩展程序
```
