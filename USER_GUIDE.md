# 福利助手 - 用户手册

## 🎁 项目简介

**福利助手**是一个基于浏览器扩展的合规福利获取助手，帮助用户更高效地获取各大平台福利，支持表单自动填充和智能提醒功能。

## 🌟 核心特性

- **🤖 智能表单自动填充** - 自动识别表单字段，提供一键填充
- **🔔 智能提醒通知** - 及时通知福利活动和最佳操作时机
- **📱 多平台支持** - 支持运营商小程序和主流电商平台
- **💾 本地化存储** - 所有数据加密存储在本地，不上传服务器
- **🛡️ 合规安全** - 完全符合平台规则，零封号风险

## 🚀 快速开始

### 安装方式

#### 方式1：自动安装（推荐）

**Windows用户:**
```cmd
git clone https://github.com/happyallday/getMoney_from_Web.git
cd getMoney_from_Web

```

**macOS/Linux用户:**
```bash
git clone https://github.com/happyallday/getMoney_from_Web.git
cd getMoney_from_Web
chmod +x install.sh
./install.sh
```

#### 方式2：手动安装

详见 [安装指南](INSTALL.md)

### 浏览器扩展安装

1. 打开Chrome/Edge浏览器
2. 访问 `chrome://extensions/`
3. 启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `browser-extension` 文件夹

## 🎯 使用指南

### 1. 首次配置

**启动后端服务:**
```bash
python python-backend/main.py
```

**配置扩展:**
1. 点击浏览器工具栏的 🎁 图标
2. 点击"设置"按钮
3. 启用需要的平台
4. 填写个人信息（可选）
5. 启用通知功能

### 2. 使用功能

**表单自动填充:**
- 访问支持的平台页面
- 在填写表单时，扩展会自动显示"🤖 自动填充"按钮
- 点击按钮即可填充预先保存的信息

**智能提醒:**
- 扩展会自动检测页面中的福利活动
- 检测到签到按钮时，显示"🤖 签到助手"按钮
- 点击按钮进行签到操作（需要确认）

**平台监控:**
- 扩展会持续监控页面变化
- 发现新的福利活动时发送通知
- 在弹窗中显示当前各平台状态

### 3. 支持的平台

#### 运营商小程序
- 📱 **中国移动** - 签到、积分兑换
- 📞 **中国联通** - 签到、优惠券领取
- 🌐 **中国电信** - 签到、话费券获取

#### 电商平台
- 🛒 **京东** - 京豆签到、优惠券领取
- 🛍️ **淘宝/天猫** - 淘金币签到、积分获取
- 🎯 **拼多多** - 签到、现金券领取

## 🐛 常见问题

### Q1: 扩展无法连接到后端？
**A:** 
- 确认Python服务正在运行
- 检查端口5000是否被占用
- 查看浏览器开发者工具的网络请求

### Q2: 表单自动填充不工作？
**A:**
- 确保已在设置中开启自动填充
- 检查是否填写了用户信息
- 刷新页面重试

### Q3: 收不到通知？
**A:**
- 确认已在设置中开启通知
- 检查浏览器的通知权限
- 确认平台已启用

### Q4: 如何卸载？
**A:**
- 停止Python服务
- 在浏览器扩展管理页面移除扩展
- 删除项目目录
